# 合并画廊区与首页精简优化方案（Unified Gallery Strip）

日期：2026-01-04

## 背景与目标

- 首页顶部信息过多（品牌标签行 + 二级标题），需精简以突出主标题与画板。
- 生成历史与灵感快照分散展示，需合并为统一画廊区以减少页面长度与多栏感。
- 首页与 /generate 统一画廊卡片样式与交互，提升一致性与可维护性。

## 结构调整

### 首页（/）

- 移除主标题上方“GLINT LAB / 温室工作台”标签行。
- 移除“高密度提示词工作台”标题，仅保留轻量提示文案。
- 合并“生成历史 + 灵感快照”为单一横向画廊区：
  - 默认展示 3–4 张卡片；超出横向滚动。
  - 卡片尺寸约 260–300px 宽，160–200px 高，保证一眼看清内容。
  - 卡片底部操作：下载 / 编辑；右上角“查看”触发 lightbox。

### /generate

- 右侧历史列表替换为同款统一画廊区。
- 维持完整工作台功能，避免重复多栏信息。

## 组件与交互

### UnifiedGalleryStrip

- 统一卡片画廊组件（首页 + /generate 共用）。
- 输入：`items`, `onPreview`, `onEdit`, `onDownload`。
- 输出：横向画廊卡片、空态、CTA。

### Lightbox 预览

- 点击“查看”弹出大图预览（Dialog）。
- 预览内提供“进入画廊”入口。
- 弹层轻量、背景磨砂，不抢主画面。

## 数据来源与策略

- 首页：优先使用 DB 最近记录（`getGenerationGalleryPage({ take: 6 })`），可与本地 history 合并去重（resultId 优先）。
- /generate：本地 history 为主；保持轻量，不强制 DB 请求。

## 视觉与材质

- 卡片边框使用 `white/70`，背景玻璃感；图片区域大面积占比。
- 操作按钮：下载为浅色描边，编辑为琥珀金渐变。
- Hover：轻抬升 + 柔光边缘，保持克制高级。

## 测试策略

- 新增 UnifiedGalleryStrip 渲染测试。
- 首页与 /generate 增加“画廊区存在性”测试。
- Lightbox 开关交互测试（查看 → 打开 → 关闭）。

## 约束

- 页面避免纵向过长；控制卡片数量与高度。
- 风格保持“温室工作台”统一。
