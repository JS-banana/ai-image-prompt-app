# 首页 UI 重设计方案（温室工作台 · 高级多彩）

日期：2026-01-04

## 目标与范围

- 目标：为生图网站首页建立“高级且富有色彩”的品牌感与生成美感，突出提示词工作台。
- 范围：仅首页（A 主视觉/品牌主张、B 生成入口、C 示例画廊/最近生成）。
- 明确不做：不新增“提示词数据看板”页面。

## 设计方向

- 方向：温室工作台（高级克制 + 高端色彩张力）。
- 气质关键词：高级、色彩张力、创意、轻盈、静奢。
- 记忆点：通栏“画布级提示词工作台” + 高级彩色渐变光层。

## 版式结构（首页）

1. 顶部极简导航：Logo + 单一 CTA（“开始生成”）。
2. 主视觉区：品牌宣言 + 副标题（短、克制），背景为柔和渐变云与轻纸纹。
3. 提示词工作台（核心）：通栏卡片（约 70% 视口宽度）。
4. 画廊区：精选条带（横向缩略图） + 本周精选主图。
5. 轻量页脚：品牌/版权/链接。

## 色彩与材质

- 基底色：象牙白、暖灰、墨黑（低饱和）。
- 重点色：苔绿、琥珀金、矿物蓝（用于主视觉与画廊高光）。
- 背景质感：
  - 细纸纹 + 压纹（极轻、低对比）。
  - 柔和径向光斑（温室顶光）。
  - 有机渐变云（非紫系，避免廉价感）。

## 字体与排版

- 标题：高反差衬线（Noto Serif SC）。
- 正文：细致无衬线（Noto Sans SC）。
- 英数字点缀（可选）：Fraunces。
- 排版节奏：留白充足、字距略拉开、行高偏松，突出高级感。

## 组件与交互

- PromptWorkbench
  - 多行输入 + 右下三枚胶囊控件（提示词库/分辨率/模型）。
  - 主按钮“生成”使用琥珀金渐变，hover 时柔光扩散。
  - 示例提示词一键填充（输入区下方轻量提示）。
- GalleryStrip
  - 横向条带缩略图，hover 仅轻微抬升 + 色彩轻增强。
- FeaturedArtwork
  - 大图卡片 + 细边框 + 微弱纸纹叠层，突出“精选”。

## 动效策略

- 入场：轻量 stagger（标题、输入区、画廊依次淡入）。
- Hover：2–4px 上浮 + 柔光叠加，节奏慢、克制。
- 动效原则：少而精，强调“高级质感”。

## 响应式策略

- 移动端：工作台全宽堆叠，按钮满宽；画廊变为横向 scroll-snap。
- 触控：hover 改为轻触高亮；保持对比度与可读性。

## 资产说明

- SVG 有机形状：用于主视觉顶光与背景层（低透明度）。
- 纹理：建议使用轻噪点或细纸纹 PNG（低不透明度）。

## 成功标准

- 视觉：高级、富有色彩且不显廉价。
- 功能：提示词输入区域显著、宽大、易用。
- 体验：首屏记忆点清晰、画廊精选感强。
